## Requirements
### Requirement: Strip whisper credit prefixes
The system SHALL remove Whisper watermark credit phrases from the beginning of transcribed text, preventing them from being injected into the active text field.

#### Scenario: French credit stripped
- **WHEN** transcribed text starts with "Sous-titres réalisés par Whisper"
- **THEN** the credit prefix is removed and only the remaining text is returned

_Tier: unit-pure — `test_text_cleaning.py`._

#### Scenario: French credit with trailing content
- **WHEN** transcribed text is "Sous-titres réalisées par Whisperbonjour"
- **THEN** the credit prefix "Sous-titres réalisées par Whisper" is removed and "bonjour" is returned

_Tier: unit-pure — `test_text_cleaning.py`._

#### Scenario: English credit stripped
- **WHEN** transcribed text starts with "Subtitles generated by Whisper"
- **THEN** the credit prefix is removed and only the remaining text is returned

_Tier: unit-pure — `test_text_cleaning.py`._

#### Scenario: No credit present
- **WHEN** transcribed text does not start with a known credit phrase
- **THEN** the text is returned unchanged

_Tier: unit-pure — `test_text_cleaning.py`._

#### Scenario: Credit with extra whitespace
- **WHEN** transcribed text is "Sous-titres réalisées par Whisper  hello world"
- **THEN** the credit prefix and any immediately following whitespace is removed

_Tier: unit-pure — `test_text_cleaning.py`._

#### Scenario: Empty result after stripping
- **WHEN** transcribed text is exactly a credit phrase with no additional content
- **THEN** the result is an empty string (or None if originally None)

_Tier: unit-pure — `test_text_cleaning.py`._

### Requirement: Strip Whisper hallucination phrases
The system SHALL remove known Whisper hallucination phrases from transcribed text wherever they occur (not only as a prefix), because Whisper emits training-corpus artifacts on silence or near-empty audio. After removal the system SHALL collapse leftover whitespace, and SHALL return an empty string when nothing meaningful remains. This is distinct from credit-prefix stripping, which only removes phrases at the start of the text.

#### Scenario: Hallucination phrase removed mid-text
- **WHEN** transcribed text contains a known hallucination phrase (e.g. "la communauté d'Amara.org") anywhere in the string
- **THEN** the phrase SHALL be removed and surrounding whitespace collapsed, leaving the remaining meaningful text

_Tier: unit-pure — `test_text_cleaning.py::test_amara_embedded_mid_text_removed`._

#### Scenario: Hallucination-only output becomes empty
- **WHEN** transcribed text consists solely of one or more known hallucination phrases
- **THEN** the result SHALL be an empty string

_Tier: unit-pure — `test_text_cleaning.py::test_amara_only_returns_empty`._

#### Scenario: Longest phrase wins over substring
- **WHEN** a hallucination phrase is a superstring of another listed phrase (e.g. "Sous-titres réalisés par la communauté d'Amara.org" vs "la communauté d'Amara.org")
- **THEN** the longest matching phrase SHALL be removed rather than leaving a partial fragment

_Tier: unit-pure — `test_text_cleaning.py`._

#### Scenario: Clean text is left intact
- **WHEN** transcribed text contains no known hallucination phrase
- **THEN** the text SHALL be returned with only normal whitespace normalization

_Tier: unit-pure — `test_text_cleaning.py::test_plain_text_unchanged`._
